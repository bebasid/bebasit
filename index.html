Tema oleh Pat Johnson

bebasit

bebasit merupakan sebuah proyek yang dibangun untuk memberikan kemudahan menggunakan aplikasi penembusan DPI untuk mengakses Netflix tanpa menggunakan VPN. Aplikasi ini sudah dirancang sedemikian rupa sehingga mudah untuk digunakan tanpa konfigurasi manual yang melelahkan. Sekali jalankan langsung bisa Neflixan.

Tanpa Mengurangi Kecepatan

Berbeda dengan VPN, bebasit tidak akan mengurangi ataupun menambah kecepatan akses internetmu.

Nonton di Indonesia, Filmnya Indonesia

bebasit tidak akan mengubah region Netflixmu menjadi region negara lain, tinggal di Indonesia, ya katalog filmnya harusnya Indonesia dong.

Bebas Virus

70 dari 71 satu aplikasi antivirus di dunia menyatakan aplikasi ini tidak terdeteksi virus, sehingga aplikasi ini aman dari virus. (Note: kemungkinan terkena false positif)

Dukungan Komunitas

bebasit dibangun menggunakan beberapa aplikasi penembus DPI yang cukup fenomenal dan dipakai oleh komunitas dari negara yang berbeda.

WINDOWS

Video Instalasi JRE

Gambar Bergerak BEBASIT

Kebutuhan Sistem

  • Windows 7 atau lebih tinggi.


Aplikasi yang diperlukan

  • 1. Google Chrome


  • 2. .NET Framework 4.6+ (Sudah terinstall secara default oleh Windows)


  • 3. Java Runtime Environment 7+ (hanya untuk metode PowerTunnel)


  • 4. Aplikasi bebasit


Prosedur Penggunaan

  • 1. Buka aplikasi bebasit.exe


  • 2. Pilih metode yang akan digunakan (Default: GoodbyeDPI + Google Chrome)


  • 3. Klik tombol "Mulai"


  • 4a. Tunggu hingga Google Chrome terbuka secara otomatis


  • 4b. Tunggu hingga log menampilkan "Berhasil membuka GoodbyeDPI/PowerTunnel"


  • 5. Selamat menonton


  • 6. Tekan tombol "SELESAI" jika ingin mematikan (jangan langsung menutup aplikasi secara langsung)


MACOS

Kebutuhan Sistem

  • MacOS 10.13 (High Sierra) atau lebih tinggi


Aplikasi yang diperlukan

  • 1. Google Chrome


  • 2. NodeJS + NPM (untuk metode GreenTunnel)


  • 3. Java Runtime Environment 7+ (hanya untuk metode PowerTunnel)


  • 4. Green Tunnel / PowerTunnel


  • 5. Aplikasi bebasid


  • Aplikasi nomor 2, 3, dan 4 akan terpasang dalam Prosedur Instalasi di bawah


Prosedur Instalasi (Green Tunnel)

  • 1. Buka terminal dan pastikan aplikasi bebasid telah terpasang


  • 2. Masukkan perintah dibawah ini


  • 3. Aplikasi akan memasang NodeJS, NPM, dan Green Tunnel


  • 4. Tunggu hingga selesai dan pastikan tidak ada error


Prosedur Instalasi (PowerTunnel)

  • 1. Buka terminal dan pastikan aplikasi bebasid telah terpasang


  • 2. Masukkan perintah dibawah ini


  • 3. Aplikasi akan memasang OpenJRE dan Green Tunnel


  • 4. Tunggu hingga selesai dan pastikan tidak ada error


Prosedur Penggunaan

  • 1. Buka terminal


  • 2a. Masukkan perintah dibawah ini (Green Tunnel)


  • 2b. Masukkan perintah dibawah ini (PowerTunnel)


  • 3. Tunggu hingga Google Chrome terbuka secara otomatis


  • 4. Selamat menonton


  • 5. Untuk berhenti, cukup tekan [Enter] di terminal

Catatan

  • 1. Terminal dapat ditutup selama masih streaming asalkan nanti diberhentikan dengan command


  • 2. Kami tidak dapat menjamin bahwa Netflix dapat diunblok (ada beberapa kasus dimana aplikasi gagal menembus DPI)
